Manual de uso da API



-Todos os campos são de preenchimento obrigatório
-Campos EMAIL, CPF, RG são UNIQUE, portanto não podem repetir.
-Para maior liberdade de alterações todos campos são CHAR no MySQL
------------------------------------------------------
Esta é uma API modelo para testes criada por Jeferson dos Santos.
Serve apenas para modelo de testes de consumo.
Aqui você pode baixar o PostMan para realizar testes.
------------------------------------------------------
